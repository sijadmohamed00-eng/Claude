# 🤖 Claude AI Interface - Mobile Ready

واجهة ويب مجانية ومتكاملة لـ Claude AI تعمل على **الجوال وسطح المكتب**. المشروع يعمل بالكامل في المتصفح باستخدام Puter.js للوصول الآمن إلى Claude API.

---

### ✨ المميزات

- ✅ **مجاني 100%:** يستخدم الطبقة المجانية من Puter.js
- 📱 **متوافق مع الجوال:** واجهة محسّنة للمس والشاشات الصغيرة
- 🎨 **وضع ليلي ونهاري:** تصميم أنيق ومريح للعين
- 📄 **دعم Artifacts:** عرض وتحميل الأكواد البرمجية
- 💬 **محادثات فورية:** ردود متدفقة مثل الموقع الرسمي
- 📂 **حفظ المحادثات:** تخزين محلي في المتصفح

---

### 📱 التشغيل على الجوال

#### الطريقة 1: GitHub Pages (الأسهل)
1. ارفع ملف `Claude.html` إلى GitHub
2. فعل GitHub Pages من الإعدادات
3. افتح الرابط في متصفح الجوال

#### الطريقة 2: سيرفر محلي
1. حمل تطبيق "Simple HTTP Server" من متجر التطبيقات
2. شغل السيرفر واختر مجلد الملف
3. افتح `http://localhost:8080/Claude.html`

---

### ⚠️ ملاحظة مهمة
**لا تفتح الملف مباشرة من الملفات!** استخدم سيرفر محلي أو GitHub Pages لتجنب مشاكل CORS والمصادقة.

---

### 🔐 المصادقة
أول استخدام سيطلب منك تسجيل الدخول إلى Puter. اسمح للنافذة المنبثقة لإكمال المصادقة.
